El Error 153 es un bloqueo impuesto por YouTube. En este caso concreto, se produce debido a una restricción aplicada por el propietario del video.

El Error 153 aparece cuando YouTube no permite la reproducción embebida de un video, aunque se cumplan todas las condiciones siguientes:

  • El video es público
  • El enlace funciona correctamente en YouTube
  • El código <iframe> es técnicamente correcto

Este comportamiento es habitual en videos promocionales oficiales, ya que Marvel / Disney suele deshabilitar la inserción.

La reproducción automática de audio está bloqueada por los navegadores web modernos.

Los navegadores impiden la reproducción automática aunque:

  • El audio existe
  • El audio MP3 es compatible
  • El elemento <audio> es correcto

Se requiere interacción del usuario para activar el audio.
